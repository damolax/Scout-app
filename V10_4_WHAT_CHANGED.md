# Scout App v10.4 — Simpler Send Emails + How to Use

## What changed

- Removed scheduled follow-up controls from the main UI.
- Due follow-ups are now simple: open Send Emails, choose the due follow-up group, click **Send Due Follow-ups Now**.
- Simplified Send Emails so most long tables are hidden by default.
- Hid Ready Leads, Preview, recent sent, due follow-up list, and saved sends unless you click **Show**.
- Removed confusing worker/cron wording from the visible Send Emails page.
- Added **How to Use** page in the left menu.
- Added `HOW_TO_USE_SCOUT_SIMPLE.md` in the package.

## No SQL required

This is a UI and guide update only.
