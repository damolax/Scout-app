# Mobile Build Status

Android APK: release builder included.

Use GitHub Actions → Build Android APK → download `scout-android-release-apk` → install `scout-release-v10.3.apk`.

If Android says this APK cannot update the old app, uninstall the previous debug APK first, then install the release APK.

iOS: wrapper source remains included, but a signed iOS IPA still requires Mac/Xcode and an Apple Developer account.
