# Scout App v10.7 — Easy Navigation + Checklist + Challenges

## What changed

- Added a Dashboard setup checklist with 12 meaningful tasks.
- Added a new Challenges page in the sidebar.
- Added 80+ challenges with progress cards and simple popups.
- Challenge cards explain exact steps when clicked.
- Dashboard now has big next-action cards.
- Auto Scout page uses simpler wording: Start finding emails, Stop, Results.
- Auto Scout results stay on the same page.
- Stop only shows while Auto Scout is running.
- Due Follow-ups now has a clear follow-up template selector.
- Follow-ups do not show schedule follow-up wording.
- How to Use now explains checklist, challenges, and follow-up template selection.
- Navigation remains simple: Home, Find Leads, Send Emails, Replies, Settings, Challenges, How to Use.

## No SQL required

This build uses existing tables and counts. It does not require a Supabase SQL patch.
