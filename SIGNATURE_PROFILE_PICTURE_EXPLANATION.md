# Signatures and Profile Pictures in Scout v8.34

## Is there a place to add a signature?

Yes. Go to:

Settings → Email Identity & Signatures

You can add:

- plain-text signature,
- optional HTML signature,
- optional profile picture URL,
- enable/disable signature appending.

Scout automatically adds the signature to:

- first outreach messages,
- follow-up messages,
- manual replies from a business conversation.

## Can Scout save the signature to Gmail?

Yes, optionally.

Click **Save + sync signature to Gmail** in Settings. This uses Gmail's Send-As signature setting. Existing Gmail connections may need to be reconnected once after v8.34 because this requires Gmail settings permission.

## Can Scout change the Gmail profile picture directly?

Not through the normal Gmail connection.

Scout can store a shared profile picture URL for identity consistency inside the app. The real Google/Gmail account profile photo must be changed manually in Google Account settings, or by a Google Workspace administrator using Admin SDK photo controls.
