# Scout App v9.4

No-cron user flow.

- Send Now sends directly while the page is open.
- Saved schedules run from the open Send Emails page.
- Keep Scout open when you want scheduled work to start.
- Auto Scout runs from the app and shows live work.
- The old cron setup is not required for this version.

## Deploy

Copy this folder into the Scout-app repo, commit, push, then redeploy Vercel without build cache.
