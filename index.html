📧 Email Scout

Upload a list of businesses with emails, write up to 20 personalised message templates using {name} as a placeholder, and send them one by one.

1. Contacts Source

No upload happens here. Contacts must come from Upload List → Verify & Send → Ready to Send. This tab is only for choosing the message, subject, signature, delay, and sending the approved batch.

2. Message Templates

Write up to 20 messages. Each message has its own subject and body. Use {name}, {email}, {category}, and other shortcodes for personalization. One message is picked per send (rotates or you select).

3. Send Settings

🌐 Bing Dorking

This is still Scout App. Use Bing signal queries to find web leads, then send them into the same queue, Verify & Send tab, and Email Scout tab.

1. Dorking Settings

Set industry, location, and Bing signal queries. These sync to the Chrome extension and can also be posted to the backend cloud runner.

2. Browser Extension Dorking

Use this when your PC and Chrome are open. The extension runs Bing dorking in the browser and pushes found web leads into Scout App.

Extension dorking uses the current saved settings.

3. Cloud Bing Dorking

Optional backend runner. It uses the same Bing signals and backend endpoints. Search engines can still block cloud scraping, so browser extension mode remains the most reliable no-API dorking mode.

No cloud campaign running.

4. Web Leads

After dorking, web leads appear in the left queue under the Web Leads filter. Then run Verify & Send to move ready contacts into Email Scout.

🧪 Verify & Send

This is a Scout App tab, not a separate app. Upload/import leads through Upload List first, then this tab checks the businesses already in your queue, blocks invalid or duplicate emails, and moves only ready contacts into the existing Email Scout tab.

1. Verify current queue

Uses the emails already found by Google Maps extraction, website crawling, dorking, or upload. It does not remove any existing Scout App feature.

2. Send ready contacts to Email Scout tab

Choose how many ready contacts to move into the existing Email Scout tab. Example: if 1000 are ready, enter 200 and only 200 go into the Email Scout send list.

🌐 Dorking Settings

Set your industry, location and signal query templates. The Elevate Scout extension reads these automatically when you start dorking. Each signal is a Bing search query with {industry} and {location} placeholders.

Each line = one Bing search query. Use {industry} and {location} as placeholders.
Examples:
"{industry}" "{location}" "just opened" "contact us"
"{industry}" "{location}" "grand opening" after:2025-01-01
site:facebook.com/pages "{industry}" "{location}" "email"
site:yelp.com "{industry}" "{location}" "new"

Your Custom Messages

Write up to 20 outreach messages of your own. Use shortcodes: {name} {rating} {reviews} {location} {industry} {website}. These are available in the Business tab under "Your Messages".
