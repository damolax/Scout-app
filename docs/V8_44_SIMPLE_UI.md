# Scout v8.44 — Simple UI

This build keeps the full feature set but makes the app easier for a first-time user.

## What changed

- Left navigation reduced to five main areas: Home, Find Leads, Send Emails, Replies, Settings.
- Leads, upload, Auto Scout, email verification, and data safety are grouped under Find Leads.
- Templates, sending, schedules, deliverability, and worker automation are grouped under Send Emails.
- Notification remains a top-right bell popup, not a sidebar page.
- Daily Scouting is not shown in navigation.
- Page headings and button labels use simpler language.
- Operations page is called Worker and uses clearer actions like Check Replies, Send Due Emails, and Find Emails.

## Kept features

- Extension ingest
- Source Scout/dorking
- Uploads
- Auto Scout email finding
- Verify/redetect/delete email controls
- Leads/business CRM
- Message templates
- Follow-up templates
- Reply-only templates
- Audience categories
- Sending now
- Scheduling later
- Durable jobs and stop button
- Sender limits and last-24h counts
- Signature and logo support
- Reply sync
- Auto-reply classifier
- Bad inbox/no-inbox detection
- Notifications bell
- Automation worker endpoint
