# Scout App v10.13 - Simple Auto Scout + Full Guide

## What changed

- Auto Scout now has only one user setting: how many leads to queue.
- Batch size, speed, and rounds are hidden and set to safe defaults automatically.
- Queue max is shown clearly: 10,000 leads per start.
- Auto Scout still uses safe internal settings: 20 checks per batch, speed 4, 3 rounds.
- Removed confusing Auto Scout controls that made the page feel technical.
- Added a full PDF user guide explaining every main page and feature in simple language.

## No SQL required

This build does not need a Supabase SQL patch.
