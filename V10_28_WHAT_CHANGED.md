# Scout App v10.28 — Follow-up + Template Clarity

## What changed

- Added a clear **Send follow-ups** shortcut at the top of Send Emails.
- Follow-ups are sent from **Send Follow-ups — 72h no reply** on the same Send Emails page.
- The main Send Emails template selector now shows **initial templates only**.
- Template rotation now rotates **initial templates only**.
- Follow-up templates no longer appear in first-message sending.
- Reply-only templates still stay out of first-message sending.
- Follow-up sending now has its own follow-up template selector.
- Settings now shows **Total sent** per Gmail account, not “sent in last 24h”.
- Seed receiver copy now explains that it receives test emails; it does not send outreach unless also used as a sender.

No SQL required.
