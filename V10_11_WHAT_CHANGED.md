# Scout App v10.11 — Stricter Scouting Levels

This build makes the Scouting Level system much harder.

## Changes

- 3,000 scouted leads alone should now keep a user around Rookie, not Strategist.
- Strategist now requires serious sustained work, not one strong scouting day.
- Scouting/import volume gives fewer points.
- Real human replies still matter most, but they no longer jump users too far too quickly.
- Replying to prospects from inside Scout remains a high-value mastery action.
- Stage 6+ is harder. Stage 11+ is intentionally very hard.
- Exact requirements remain hidden; users only see hints.

No SQL required.
